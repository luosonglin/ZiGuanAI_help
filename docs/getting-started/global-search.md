# 全局搜索使用

系统提供强大的全局搜索功能，帮助您快速定位所需信息。

## 搜索入口

### 顶部搜索框

在页面顶部导航栏右侧有搜索框，输入关键词即可搜索。

### 搜索功能

- **全文搜索** - 搜索所有文档内容
- **高级搜索** - 按类别、时间等条件筛选
- **搜索历史** - 记录最近搜索记录

## 使用技巧

1. 输入关键词后按回车键搜索
2. 使用空格分隔多个关键词
3. 支持中英文混合搜索
