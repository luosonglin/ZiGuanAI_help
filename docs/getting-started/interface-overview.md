# 界面总览

本文介绍系统界面的布局和主要功能区域。

## 界面布局

### 顶部导航栏

包含系统logo、标题、导航菜单和搜索框。

### 左侧侧边栏

按功能模块分类的导航菜单，支持展开/折叠。

### 主内容区

当前操作的主要区域，展示文档内容或操作界面。

## 功能按钮

- **返回顶部** - 快速回到页面顶部
- **主题切换** - 切换明暗主题
- **用户菜单** - 用户信息和设置
