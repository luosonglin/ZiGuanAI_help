# 账号登录与权限

本文介绍系统账号的登录方式和权限管理相关内容。

## 登录方式

### 账号密码登录

1. 访问系统登录页面
2. 输入用户名和密码
3. 点击登录按钮

### 权限说明

系统采用角色权限管理，不同角色拥有不同的操作权限：

| 角色 | 权限说明 |
|------|----------|
| 管理员 | 系统全部功能权限 |
| 部门负责人 | 部门数据管理权限 |
| 普通用户 | 基础查询和操作权限 |

## 密码安全

- 建议定期更换密码
- 密码长度不少于8位
- 包含数字和字母组合
