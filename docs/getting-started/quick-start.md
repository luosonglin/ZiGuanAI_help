# 系统快速上手

欢迎使用科学城集团数智化资管系统！本指南将帮助您快速了解系统的核心功能和基本操作。

## 系统概览

数智化资管系统是一套面向资产管理的综合性平台，提供以下核心功能：

### 主要模块

1. **资管工作台** - AI驱动的智能助手，支持报告自动生成
2. **投资企业档案** - 企业信息管理和档案归档
3. **企业经营监测** - 实时经营数据监控和分析
4. **报告工作台** - 标准化报告制作和导出
5. **资管数据库** - 财务、法规、政策等数据资源库
6. **资管内训** - 在线学习和知识分享平台

## 快速导航

### 第一步：登录系统

访问系统登录页面，使用您的账号密码登录。

### 第二步：了解界面

系统采用三栏布局：
- **顶部导航** - 主要功能入口和搜索
- **左侧侧边栏** - 功能模块导航
- **主内容区** - 当前操作区域

### 第三步：开始使用

根据您的需求选择相应的功能模块开始操作。
