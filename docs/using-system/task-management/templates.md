# 智能任务框架

基于预设的业务逻辑与交互框架，快速创建标准化的股权投后管理任务。系统正逐步升级为更智能的AI Skill，实现从机械填报到智能服务的跨越。

## 功能概述

智能任务框架专为股权投后管理业务设计，帮助您高效完成各类投后管理工作：

- **企业评分评级**：定期对投资企业进行综合评分和等级评定
- **财务数据报送**：按集团统计口径收集和上报企业财务数据
- **风险评估**：对投资企业进行定期风险评估
- **重大事项报告**：处理和上报企业重大事项

## 使用智能任务框架

### 操作步骤

1. **打开任务创建页面**


2. **选择任务类型**
   - 点击表单底部中的「选择任务类型」按钮
   - 打开任务类型选择弹窗
![选择任务类型](https://cdn.songlin.site/工作/资管项目/20260525162616931.png)


3. **筛选任务类型**
   - 通过分类标签筛选：评分评级档案卡、财务报告框架等
   - 使用搜索框输入关键词搜索
![筛选任务类型](https://cdn.songlin.site/工作/资管项目/20260525162707053.png)


4. **预览与应用**
   - 点击任务类型卡片查看详细内容
   - 确认后点击「应用」按钮
   - 任务表单自动填充预设内容
![应用任务类型](https://cdn.songlin.site/工作/资管项目/20260525162745362.png)


### 应用效果示例

应用任务类型后，表单自动填充：
- 任务标题（含类型名称和周期信息）
- 任务描述（含执行说明和要求）
- 任务分类和优先级（按预设配置）
- 关联表单和参考文档

## 任务框架管理

### 框架库界面

![框架库](https://cdn.songlin.site/工作/资管项目/20260525162707053.png)

框架库提供以下功能：
- **分类浏览**：按业务类型分类展示任务框架
- **搜索过滤**：支持关键词搜索和条件筛选
- **收藏管理**：收藏常用框架便于快速访问
- **使用统计**：查看框架使用频次和反馈

### 创建自定义任务框架

1. 点击框架库右上角「新建框架」
2. 填写框架基本信息：名称、分类、适用范围
3. 设计框架内容：标题模板、描述模板、表单字段
4. 配置框架参数：默认优先级、截止时间规则、通知设置
5. 上传相关附件：参考文档、表单模板等
6. 保存并发布框架

### 框架版本管理

- **版本记录**：自动保存框架修改历史
- **版本对比**：查看不同版本间的差异
- **版本回滚**：恢复到历史版本
- **版本发布**：控制框架生效时间


### 内容规范要求
- 严格按照预设格式填写内容
- 财务数据需符合集团统计口径
- 评分评级需依据统一标准执行
- 附件需按要求命名和格式上传

### 框架更新管理
- 定期关注框架更新通知
- 及时使用最新版本框架
- 反馈框架使用问题和改进建议

## 向AI Skill演进

### 演进背景

当前的任务框架系统正在逐步转型为更智能的AI Skill，这是系统智能化升级的重要战略方向。

### 核心理念转变

从传统框架到AI Skill，代表着从「机械填报」到「智能服务」的根本性转变：

| 维度 | 传统框架 | AI Skill（演进方向） |
|------|---------|-------------------|
| **核心定位** | 任务框架库 | 智能任务助手 |
| **交互方式** | 表单填写 | 自然语言交互 |
| **处理模式** | 被动接收 | 主动理解 |
| **响应方式** | 机械校验 | 智能生成 |
| **学习能力** | 无 | 持续进化 |

### 演进路径

系统正沿着以下路径逐步演进：

#### 第一阶段：标准化框架（当前）
- 提供标准化的任务框架和表单
- 确保数据规范和流程统一
- 建立完善的框架管理体系

#### 第二阶段：AI增强（进行中）
- 在框架基础上增加AI智能推荐功能
- AI辅助填充数据和生成内容
- 逐步引入自然语言理解能力

#### 第三阶段：AI Skill（目标）
- 完全智能化的任务处理能力
- 支持自然语言交互和智能决策
- 实现真正的智能投后管理

### 当前已实现的AI能力

虽然当前仍以框架为主要形式，但系统已逐步融入AI能力：

1. **智能推荐**：根据任务类型和企业特征推荐合适的框架
2. **自动填充**：基于企业档案自动填充基础信息
3. **数据校验**：AI辅助的数据质量检查和格式校验
4. **智能提示**：根据上下文提供操作建议和帮助

### 未来AI Skill特性

随着演进的深入，AI Skill将具备以下核心能力：

- **动态理解**：能够理解用户的自然语言描述，自动解析需求
- **智能响应**：根据用户输入智能生成个性化的任务内容
- **主动服务**：基于历史数据和上下文主动提供服务
- **学习进化**：通过持续学习不断优化处理逻辑
- **场景适配**：自动适应不同业务场景和用户需求

### 过渡期间的使用建议

在从传统框架到AI Skill的过渡期间：

1. **熟悉框架系统**：掌握当前框架的使用方法，这是AI Skill的基础
2. **关注AI增强功能**：留意系统新增的AI辅助功能
3. **反馈使用体验**：您的反馈对AI能力的优化至关重要
4. **逐步适应新交互**：尝试使用AI推荐等新功能，适应智能化操作方式
