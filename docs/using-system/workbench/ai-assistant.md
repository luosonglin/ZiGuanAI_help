# AI资管助手

AI资管助手是系统的智能助理，提供智能化的股权投后管理支持，帮助您高效完成各项投后管理任务。

## 功能介绍

### 智能问答

支持自然语言提问，快速获取答案：
- 投后管理政策咨询
- 企业档案信息查询
- 财务数据分析
- 风险评估建议

### 报告生成

自动生成标准化报告，节省时间成本：
- 投后检查报告
- 企业评分评级报告
- 财务分析报告
- 风险评估报告

### 数据分析

提供数据分析和趋势预测：
- 财务数据趋势分析
- 行业对标分析
- 投资回报分析
- 风险预警分析

### AI模板辅助

AI智能辅助任务模板管理，提升工作效率：

#### AI智能推荐模板

根据任务类型、企业特征、历史数据，AI自动推荐最合适的模板：
- 分析任务类型和目标
- 识别企业行业、规模、发展阶段
- 参考历史任务执行情况
- 匹配集团最新政策和标准
- 提供匹配度评分和推荐理由

**使用方式**：
1. 在任务创建页面，点击「AI推荐模板」
2. 输入任务描述，例如："对XX科技进行季度综合评分"
3. AI分析后推荐3-5个候选模板
4. 查看推荐理由和匹配度评分
5. 选择最合适的模板应用

#### AI自动生成模板

基于自然语言描述，AI快速生成符合规范的任务模板：

**自然语言生成示例**：
- **输入**："为制造业企业创建年度综合评分模板，重点关注财务指标和经营能力"
- **AI生成**：自动创建包含财务指标（营收增长率、净利润率等）和经营能力（市场份额、客户满意度等）维度的评分模板

- **输入**："创建符合集团2024年统计口径的季度财报模板"
- **AI生成**：自动匹配最新统计口径，生成资产负债表、利润表、现金流量表模板

**历史任务学习**：
AI学习历史优秀任务，自动生成优化模板：
- 分析历史任务执行效果
- 识别最佳实践和常见问题
- 生成改进后的模板版本

#### AI智能填充

AI自动填充企业信息和关键数据：

**自动填充企业信息**：
- 企业名称、企业代码
- 所属行业、企业规模
- 投资时间、投资金额
- 历史评分评级结果

**自动填充评估参数**：
- 评估周期：根据任务类型自动计算
- 统计口径：自动匹配最新版本
- 评分标准：根据企业类型调整
- 截止时间：根据工作日历智能推荐

**自动填充参考数据**：
- 历史财务数据
- 行业平均水平
- 同类企业对比数据
- 风险等级参考

#### AI模板优化

AI分析历史任务执行情况，提供模板优化建议：

**执行效果分析**：
- 任务完成率
- 数据质量评分
- 用户反馈汇总
- 问题识别和改进建议

**智能优化建议**：
- 调整评分维度和权重
- 优化数据采集字段
- 改进流程和步骤
- 增加质量控制点

## 使用方法

### 智能问答使用

1. 在资管工作台中找到AI助手入口
2. 输入您的问题或需求
3. 系统自动生成回答或建议

### 报告生成使用

1. 选择报告类型
2. 输入关键参数
3. AI自动生成报告初稿
4. 人工审核和调整

### AI模板辅助使用

1. 在任务创建页面，点击「AI推荐模板」或「AI生成模板」
2. 输入任务描述或需求
3. AI推荐或生成模板
4. AI自动填充关键数据
5. 确认后创建任务

| `{企业代码}` | 企业在集团的统一编码 | ENT-2024-001 |

| `{评估周期}` | 当前评估周期 | 2024年Q1 |
| `{统计口径}` | AI匹配的最新统计口径版本 | V2024.03 |

## 注意事项

### AI推荐使用建议
- 优先使用AI推荐的模板，匹配度更高
- 仔细阅读AI的推荐理由，了解推荐依据
- 如AI推荐不合适，可手动选择其他模板
- 及时反馈AI推荐效果，帮助AI持续优化

### AI生成质量保证
- AI生成后务必人工审核
- 确认模板内容符合集团规范
- 根据实际情况调整模板内容
- 重要任务建议使用标准模板

### 数据准确性
- AI填充的数据需核对准确性
- 关键数据建议人工确认
- 发现数据错误及时反馈
- 定期更新企业基础信息

## 常见问题

### Q：AI推荐的模板一定是最合适的吗？
A：AI推荐基于多维度分析，匹配度较高，但仍需人工确认。建议查看AI的推荐理由，结合实际情况选择最合适的模板。

### Q：AI生成的模板可以使用吗？
A：AI生成的模板可以参考使用，但务必人工审核确认。重要任务建议使用经过验证的标准模板。

### Q：如何让AI推荐更准确？
A：提供详细的任务描述，包括任务目标、企业特征、时间要求等信息，AI分析越充分，推荐越准确。

### Q：AI会学习我的使用习惯吗？
A：是的，AI会学习您的历史任务和模板使用习惯，随着使用次数增加，推荐会越来越精准。
