# 常见问题FAQ

这里收集了用户最常问的问题。

## 登录问题

### 忘记密码怎么办？

点击登录页面的"忘记密码"链接，按照提示重置密码。

### 账号被锁定怎么办？

联系系统管理员解锁账号。

## 使用问题

### 如何生成报告？

在资管工作台中选择报告模板，填写相关信息即可生成。

### 如何搜索内容？

使用页面顶部的搜索框输入关键词搜索。

## 技术问题

### 系统响应慢怎么办？

请检查网络连接，或联系技术支持。

### 无法上传文件怎么办？

检查文件格式和大小是否符合要求。
